# Phase 2 Realms — Expansion Overview

**Category:** World — Future Content
**Status:** 📋 Design Phase (Not In Scope for MVP)

---

## 1. Overview

Phase 2 expands the Nine Realms from 3 (Asgard, Nidavellir, Alfheim) to 9. Each realm adds a unique resource, chaos type, and environmental mechanic. Phase 2 realms are **teased** in Phase 1 via locked Bifrost destinations and "useless" item drops from Proxies.

---

## 2. Realm Briefs

### Muspelheim (Fire) 🔥
- **Vibe:** Volcanic heat, rivers of lava, obsidian plateaus
- **Resource:** Cinder-Wheat (heat-resistant crop)
- **Chaos State:** Eruption — lava flows consume walkable tiles
- **Unique Mechanic:** Temperature control via Cooling Totems

### Svartalfheim (Dark Elves) ⚙️
- **Vibe:** Dimly lit underground clockworks, shadow-gem mines
- **Resource:** Shadow-Gems (precision components)
- **Chaos State:** Gear Jam — clockwork machinery locks up
- **Unique Mechanic:** Light/dark puzzle mechanics; visibility based on lamp placement

### Vanaheim (Vanir) 🌧️
- **Vibe:** Lush rain-heavy wetlands, marshes, ancient druidic stones
- **Resource:** Wild Rot (fermented organic compound)
- **Chaos State:** The Flood — rising water levels submerge nodes
- **Unique Mechanic:** Weather cycles affect harvest timing

### Jotunheim (Giants) 🏔️
- **Vibe:** Massive scale, imposing ice cliffs, eternal fog
- **Resource:** Frost-Core (giant crystallized ice energy)
- **Chaos State:** Eternal Fog — visibility reduced to near-zero
- **Unique Mechanic:** Scale puzzle — Odin is tiny relative to environment

### Helheim (Underworld) 💀
- **Vibe:** Gloomy mist, archive of forgotten things, ghostly echoes
- **Resource:** Memory-Dust (extracted from lost artifacts)
- **Chaos State:** The Silence — all ambient sound stops, nodes become invisible
- **Unique Mechanic:** "Memory" system — revisiting spots reveals past events

### Midgard (Human Realm) 🏡
- **Vibe:** Pastoral village, out-of-place modern touches
- **Resource:** Mundane Materials (wood, stone — simple but versatile)
- **Chaos State:** Portal Instability — reality glitches
- **Unique Mechanic:** Features a bewildered human "vibe coder" NPC who accidentally opened a local portal

## 3. Phase 1 Tease Mechanics

### Locked Bifrost Destinations
- Phase 2 realms visible on the Bifrost UI with error overlays.
- **Jotunheim:** Covered in unbreakable ice.
- **Muspelheim:** Glowing with "Too Hot" warning.
- **Others:** "Signal Lost" static overlay.

### Teaser Item Drops
Proxies occasionally dig up "useless" items from Phase 2 realms:
- **Cinder-Seeds** from Nidavellir mining → "Needs volcanic heat. Useless here."
- **Shadow Shards** from Alfheim harvesting → "Too dark to identify."
- These items stack in the Vault but have no function until Phase 2 unlocks.
